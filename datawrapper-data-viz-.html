Datawrapper - data viz tool

September 18, 2021 â€¢ Vibek Raj Maurya

Initially intended to reporters as a tool for data journalism, Datawrapper fills a niche with its ability to enrich stories with charts, maps and tables that are interactive, responsive, and embeddable on any website.

Datawrapper's free tier is very generous with various live-updating charts, easy to embed and responsive. The professional solution is expensive but allows removing the Datawrapper attribution and create print-ready graphics in SVG and PDF version. The company is based in Berlin.

Below are the auto-updating graphs generated using Datawrapper. The data for the graph are served directly from github that gets updated daily.
